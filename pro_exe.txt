	<h1>HTML<br/> [HyperText Markup Language]</h1>
	<p><strong>HTML</strong>은 전자 문서의 서식을 정의하기 위해 만들어졌으며, 국제표준 SGML의 부분 집합으로 정의되었다. <strong>HTML</strong>은 SGML에서 특히 하이퍼텍스트를 강조하여 만들어진 언어이며, ASCII 문자로 구성된 일반적인 텍스트로 구성되었다. 이 언어는 별도 컴파일러가 필요치 않으며, 웹 브라우저에서 해석이 가능한 사용하기 쉬운 언어로 각광을 받고 있다.<a href="http://www.webcss.kr/">[ webcss ]</a></p>
</div><!--text_content-->
<div class="text_content">
	<h1>CSS<br/> [ Cascading Style Sheets ]</h1>
	<p>웹 문서의 전반적인 스타일을 미리 저장해 둔 스타일시트이다. 문서 전체의 일관성을 유지할 수 있고, 세세한 스타일 지정의 필요를 줄여 이용에 편리성을 더 했다.<br/> 기존의 웹 언어인 <strong>HTML</strong>은 웹 문서를 다양하게 설계하고 수시로 변경하는데 많은 제약이 따랐다. 이를 보완하기 위해 만들어진 것이 스타일 시트이고 스타일 시트의 표준안이 바로 <strong>CSS</strong>이다. 간단히 스타일시트라고도 한다. <a href="http://www.webcss.kr/"> [ webcss ]</a></p>
</div><!--text_content-->
<div class="text_content">
	<h1>검색 엔진 최적화<br/> [ SEO, search engine optimization]</h1>
	<p>검색 엔진에서 검색했을 때 상위에 나타나도록 관리하는 것. 검색 결과 상위에 링크되는 것이 유리하다. 검색 엔진 최적화<strong> [ SEO, search engine optimization]</strong> 방법으로는 자신의 사이트를 대표할 수 있는 핵심적인 키워드를 두세 개 선택해 이를 사이트 등록에 사용하고, 이를 배너 교환이나 여러 사이트에서 추천 사이트로 등록할 경우 자연히 이용자가 많아지고, 이용자가 늘어나면 사이트의 순위도 올라가게 된다. <a href="http://www.webcss.kr/"> [ webcss ]</a></p>
